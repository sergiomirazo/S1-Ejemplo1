# API REST CRUD - Semana 1 | Sistemas de Información UNIDEP

## Descripción
API básica desarrollada en Python con Flask que expone operaciones CRUD sobre una base de datos simulada en memoria (diccionario). Diseñada como actividad introductoria para comprender el ciclo de vida de una API REST.

## Requisitos
- Python 3.8+
- Flask (`pip3 install flask`)

## Instalación y ejecución
1. Clona o descarga este proyecto.
2. Crea un entorno virtual e instala las dependencias:
   ```bash
   python3 -m venv venv
   source venv/bin/activate  
   pip3 install flask
